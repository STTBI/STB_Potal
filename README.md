# STB_Portal
스파르타 내일배움캠프에서 플랫포머 퍼즐게임 포탈1을 레퍼런스로 진행한 팀 프로젝트입니다.

</div>
  
----
  
## 📌 프로젝트 개요
  
- **프로젝트**: 포탈 모작
- **개발환경**: Unity, C#  
- **개발인원**: 5
- **주요기능**:  이동:wasd, 포탈 설치: 마우스 좌,우 클릭, 큐브 잡기: E, 슬로우 모션: F
